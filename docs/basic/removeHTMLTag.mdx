# removeHTMLTag

> `removeHTMLTag` 是一个将所有 HTML 标签从一个字符串中删除的函数

### :bento: 参数

1. `string` - 待处理的字符串

### :fire: 返回值

`string` - 删除所有 HTML 标签后的字符串

### :rocket: 示例

```jsx
import { removeHTMLTag } from 'atools-js';

const str = '<p>Hello <b>World</b></p>';

const result = removeHTMLTag(str);
console.log(result); // 'Hello World'
```

### :bulb: 源码

:::tip source code

```js title="/atools/_basic/removeHTMLTag.ts"
export const removeHTMLTag = (str: string): string => str.replace(/<[^>]+>/g, '');
```

:::
