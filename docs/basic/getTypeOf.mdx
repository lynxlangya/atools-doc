# getTypeOf

> `getTypeOf` 用于获取一个值的类型，返回值为字符串。

### :bento: 参数

1. `any` - 可以是任何类型的值。

### :fire: 返回值

`string` - 返回值为字符串。

- `string`
- `number`
- `boolean`
- `symbol`
- `null`
- `undefined`
- `object`
- `array`
- `regexp`
- `date`
- `error`
- `function`
- `asyncfunction`
- `htmldocument`

### :rocket: 示例

```js
import { getTypeOf } from 'atools-js';

getTypeOf(1); // number
getTypeOf(true); // boolean
getTypeOf(Symbol()); // symbol
```

### :bulb: 源码

:::tip source code

```js title="/atools/_basic/getTypeOf.ts"
export const getTypeOf = (param: unknown): string => {
  const type = Object.prototype.toString.call(param).slice(8, -1);
  return type.toLowerCase();
};
```

:::
