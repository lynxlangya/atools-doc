# is

> `is` 类型断言, 可以用来确定一个值的类型

### :bento: 参数

1. `any` - 需要断言的参数
2. `string` - 需要对比的类型

### :fire: 返回值

`boolean`

### :rocket: 示例

```js
import { is } from 'atools-js';

is(1, 'number'); // true

is(1, 'string'); // false

is(new Date(), 'date'); // true
```

### :bulb: 源码

:::tip source code

```js title="/atools/_basic/is.ts"
// highlight-next-line
import { getTypeOf } from './getTypeOf';

export const is = (value: any, type: string): boolean => {
  return getTypeOf(value) === type.toLowerCase();
};
```

:::
