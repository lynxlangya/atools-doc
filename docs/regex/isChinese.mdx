# isChinese

:::note
`isChinese` 是否中文/汉字
:::

### :bento: 参数

1. `string` - 待判断的值

### :fire: 返回值

`boolean` - 判断结果

### :rocket: 示例

```jsx title="isChinese"
// highlight-next-line
import { isChinese } from 'atools-js';

isChinese('中文'); // true
isChinese('Hello'); // false
```
