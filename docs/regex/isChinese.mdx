# isChinese

> `isChinese` 是否中文/汉字

### :bento: 参数

1. `string` - 待判断的值

### :fire: 返回值

`boolean` - 判断结果

### :rocket: 示例

```jsx
import { isChinese } from 'atools-js';

isChinese('中文'); // true
isChinese('Hello'); // false
```

### :bulb: 源码

:::tip source code

```js title="/atools/_regex/isChinese.ts"
export const isChinese = (str: string): boolean => {
  const reg = /^[\u4e00-\u9fa5]+$/;
  return reg.test(str);
};
```

:::
