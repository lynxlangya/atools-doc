# isUrl

:::note
`isUrl` 判断是否是一个 URL
:::

### :bento: 参数

1. `string` - 待判断的值

### :fire: 返回值

`boolean` - 判断结果

### :rocket: 示例

```jsx title="isUrl"
// highlight-next-line
import { isUrl } from 'atools-js';

isUrl('http://www.baidu.com'); // true
isUrl('https://www.baidu.com'); // true
isUrl('baidu.com/index.html?a=1&b=2#hash'); // true
isUrl('baidu')  // false
```
