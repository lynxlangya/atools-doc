# isBrowser

> `isBrowser` 判断当前环境是否是浏览器环境

### :bento: 参数

- `无`

### :fire: 返回值

`boolean` - 当前环境是否是浏览器环境

### :rocket: 示例

```jsx
import { isBrowser } from 'atools-js';

console.log(isBrowser);
```

### :bulb: 源码

:::tip source code

```js title="/atools/_browser/isBrowser.ts"
export const isBrowser: boolean =
  typeof window === 'object' && typeof document === 'object';
```

:::
