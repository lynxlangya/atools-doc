## 轻松上手

```bash
# npm
npm install atools-js

# yarn
yarn add atools-js

# pnpm 🔥
pnpm add atools-js
```

## 源码获取

```bash
# clone
git clone git clone git@github.com:wangdaoo/atools.git

# enter
cd atools

# install
pnpm/npm/yarn install
```
