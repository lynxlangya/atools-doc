# 介绍

<Image img="http://cdn.wangdaoo.com/atools_banner.png" />

:::tip
`ATools` 是由 `TypeScript` 编写，`rollup`打包的一款现代 `js` 函数类库
:::

## 轻松上手

```bash
# npm
npm install atools-js

# yarn
yarn add atools-js

# pnpm 🔥
pnpm add atools-js
```

## 源码获取

```bash
# clone
git clone git clone git@github.com:wangdaoo/atools.git

# enter
cd atools

# install
pnpm/npm/yarn install
```
